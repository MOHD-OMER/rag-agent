Features
📑Multi-Document Ingestion
Supports PDF, TXT, DOCX with auto-chunking (512 tokens, 64 overlap). Recursive character splitter preserves context.
pypdfpython-docxunstructured
🔀Hybrid Retrieval
BM25 (sparse, w=0.4) + ChromaDB (dense, w=0.6) combined via EnsembleRetriever. +24% MRR@5 vs dense-only.
rank-bm25chromadbEnsembleRetriever
🧩LangGraph Agent
ReAct loop with 3 tools. Conditional edges route between agent and tool execution. Max 10 recursion steps.
langgraphReActtool_calls
💬Streaming Responses
Server-Sent Events (SSE) stream tokens in real-time. Tool activity events surfaced to the UI as they happen.

Quick Start
1
Install dependencies
pip install -r requirements.txt
2
Configure environment
cp .env.example .env # Set GROQ_API_KEY and TAVILY_API_KEY
3
Start FastAPI backend
cd backend && uvicorn main:app --reload --port 8000
4
Start Streamlit UI
streamlit run frontend/app.py --server.port 8501
5
Or use Docker Compose (one command)
docker compose up --build
Starts backend (:8000), Streamlit (:8501), and Gradio (:7860) simultaneously.
